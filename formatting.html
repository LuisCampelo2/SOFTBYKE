Formatação de Texto

Exemplos de todos os elementos HTML de formatação com contexto prático da SOFTBYKE

<strong> — Negrito

Neste fim de semana, a SOFTBYKE oferece 20% de desconto em todas as mountain bikes. Não perca!

Use <strong> para marcar texto como semanticamente importante. O navegador o exibe em negrito.

<em> — Itálico (Ênfase)

A SOFTBYKE vende bicicletas de qualidade premium, escolhidas com cuidado e expertise dos melhores fabricantes do mundo.

Use <em> para enfatizar texto. Diferente de <i> — <em> tem significado semântico.

<u> — Sublinhado

Nossa loja fica na SHC/SW Quadra 302, Bloco A, Sudoeste, Brasília.

Use <u> com moderação — em alguns idiomas para nomes próprios ou para destacar termos específicos. Evite em links para não confundir o usuário.

<s> — Tachado

Road Racer Carbon — R$ 5.800,00 R$ 5.199,00 — promoção somente esta semana!

Use <s> para indicar que uma informação não é mais válida — ótimo para exibir preços antigos em promoções.

<sup> — Sobrescrito

A fórmula da pressão dos pneus: P = F/A (força dividida pela área, medida em kgf/cm2). A SOFTBYKE é uma das lojas mais bem avaliadas de Brasília desde 2018*.

Use <sup> para expoentes, marcas de nota de rodapé e notações matemáticas.

<sub> — Subscrito

Ciclistas se hidratam com H2O. Nossa bebida esportiva contém NaCl (cloreto de sódio) para repor eletrólitos após um longo esforço que consome O2.

Use <sub> para fórmulas químicas, referências de notas de rodapé e subscritos matemáticos.

<h1> a <h6> — Títulos

h1 — Título Principal da SOFTBYKE

h2 — Categoria de Produto

h3 — Nome da Seção

h4 — Subseção

h5 — Rótulo de Detalhe
h6 — Menor nível de título

Use apenas um <h1> por página (boas práticas de SEO). Use os títulos em ordem para criar uma estrutura lógica de documento.

<hr> — Linha Horizontal

Mountain Bikes — a partir de R$ 1.200,00

Bikes de Estrada — a partir de R$ 2.800,00

Bikes Urbanas — a partir de R$ 900,00

Use <hr> para separar seções temáticas de conteúdo. É uma tag autofechante — não precisa de tag de fechamento.

<blockquote> — Citação em Bloco
"A vida é como andar de bicicleta — para manter o equilíbrio, você precisa continuar se movendo."
— Albert Einstein
"Toda vez que vejo um adulto andando de bicicleta, deixo de desesperar pelo futuro da raça humana."
— H.G. Wells

Use <blockquote> para citações de fontes externas. Geralmente recuado e estilizado de forma diferente do texto do corpo.
