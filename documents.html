Documentos

Wireframes, descrições das páginas e links do projeto

Wireframes das Páginas

Cada card abaixo apresenta um wireframe simplificado e a descrição de cada página do projeto SOFTBYKE. Os wireframes representam a estrutura do layout — não o estilo visual final.

index.html — Início

A página inicial é a primeira impressão da SOFTBYKE. Contém um banner hero em tela cheia com botão de chamada para ação, uma seção de apresentação com a missão e os diferenciais da loja, e uma seção de "Produtos em Destaque" com três cards de produtos.

BANNER HERO
Ver Página

products.html — Produtos

Exibe o catálogo de bicicletas usando os três tipos de listas HTML: lista não ordenada (<ul>) para os modelos disponíveis, lista ordenada (<ol>) para os mais vendidos e um guia de compra, e lista de definições (<dl>) para o glossário técnico. Esta página demonstra o uso semântico correto das listas em HTML.

Ver Página

gallery.html — Galeria

Galeria com 25 fotos de bicicletas organizadas em CSS Grid. Desktop: 3 colunas. Tablet: 2 colunas. Mobile: 1 coluna. Todas as imagens têm a mesma altura fixa (220px) com object-fit: cover para consistência visual. Efeito de hover com borda verde ao passar o mouse.

Ver Página

company.html — Empresa (Briefing)

A página de briefing responde às 18 perguntas que definem o escopo do projeto, o público-alvo, os objetivos, o orçamento e as decisões técnicas. Cada resposta é exibida em um card estilizado com borda esquerda verde para criar hierarquia visual.

Ver Página

contact.html — Contato

Layout em duas colunas: informações de contato da loja à esquerda (telefone, e-mail, endereço, horários) e formulário de contato à direita com campos de nome, e-mail, telefone, assunto (select) e mensagem (textarea). O formulário usa validação nativa HTML5 com o atributo required.

Ver Página

anchors.html — Links de Âncora

Demonstra o uso de âncoras internas. Um menu de navegação no topo contém 14 links que saltam para 14 seções na mesma página. Cada seção tem um atributo id único e aborda um tema relacionado ao ciclismo.

Ver Página

formatting.html — Formatação de Texto

Demonstra todas as tags de formatação HTML: <strong>, <em>, <u>, <s>, <sup>, <sub>, <h1>–<h6>, <hr> e <blockquote>. Cada elemento é exibido em um box com rótulo e exemplo prático usando conteúdo da SOFTBYKE.

Ver Página

links.html — Links Externos

Grid com 2 colunas contendo 10 cards de links externos, cada um apontando para um site real de marca de bicicletas. Todos os links abrem em nova aba (target="_blank"). Cada card inclui o nome da marca, uma descrição curta e um botão com ícone Lucide external-link.
